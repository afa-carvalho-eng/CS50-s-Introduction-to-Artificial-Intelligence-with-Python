# 🌀 Maze Solver
Solve a maze using different search algorithms! 🏁

## 🏃‍♂️ Running the Maze Solver
### 🔍 Breadth-First Search (BFS):
        python maze.py maze.txt bfs
### 🌲 Depth-First Search (DFS):
        python maze.py maze.txt dfs
